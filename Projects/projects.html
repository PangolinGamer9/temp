I created this calculator within the Desmos Graphing calculator. I created this because
I had an interest in creating artwork within a calculator. This was the first piece of
art that I made in Desmos. The graph is embedded, so you can interact with the calculator,
and if you click on edit graph, it will take you to Desmos' website.

This is a website that I created in my freetime for a game called Strategy of War. It
is a game that is run by a couple of my friends, but the game has a lot of statistics
that make it very difficult for the players to visualize the data they are give because
everything is text based. The purpose of this website is to make it easier for players
to understand what is happening in the game. The image is linked, so if you click it,
it will take you to the website. This website is currently a work-in-progress.

This is another graph that I created in Desmos. This one is an interactive activity. Users
can click shuffle, and then they must arrange the middle 3 by 3 grid to match the goal.
Users can slide pieces into the empty tile by clicking any of the colors around the black
tile. Once users have completed the puzzle, they may check, to determine if they are correct.
This game is still a work-in-progress because some of the colors look very similiar, and
the buttons can malfunction.

This is a website that I created to showcase the Chat-GPT 3 model. It uses context and information
from field civil-engineers to create a draft paragraph to be submitted to their superiors for review.
I created this website to help my father showcase the AI model, and its potential use cases within
his job. You can also click on the image and it will take you to the website; however, to use it
you will need to get an API key, this is to prevent people from stealing my API key because
the website is opensource.
